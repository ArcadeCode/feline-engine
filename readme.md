## 🎮 Introduction

**FelineEngine** is a lightweight, custom 2D game engine created by **Esteban Dervaux**, tailored for developing games with a top-down perspective — especially complex simulation games like *Factorio*.

This engine is currently being developed to power [**Colonia**](https://github.com/ArcadeCode/colonia) — a factory-building game currently in progress.

It is built using **C++17** and **SDL3**, focusing on performance, simplicity, and ease of prototyping for indie-scale games.

## 🌹 Philosophy
FelineEngine is built with the following principles in mind:
- ✨ Minimalism and readability first
- 🔁 Highly modular and maintainable codebase
- 🎯 Specifically suited for 2D top-down simulation and builder games
- 🚀 Designed for long-term scalability (ECS, tilemaps, dynamic entities)
