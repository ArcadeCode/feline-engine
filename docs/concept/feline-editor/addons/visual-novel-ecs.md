# Visual Novel ECS Pack

Visual Novel ECS Pack is an addons who give a complete Visual Novel editor & game engine tools. This include :
- Using the Feline Engine `Ressource<Localization>` to add i18n to your games.
- Scenes optimization with picture background and pnj sprites.
- Graph editor of all possibles choices.

## Why adding a Visual Novel ECS Pack has an official pack ?
- Because some friend of mine wanted to created a visual novel game, so, I will give them the opportunity.
- I've never thinking about how visual novels games work, so I get some answers with this projet.
