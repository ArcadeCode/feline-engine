# Tilemap ECS Pack

Tilemap ECS Pack is an addons who give a complete Tilemap & tileset editor & game engine tools. This include :
- Using the Feline Engine `Ressource<Atlas>` and offering a complete abstracted `Ressource<TileMap>`
- High optimization of tile update.

## Why adding Tilemap ECS Pack has an official pack ?
- To create my own game based on Feline Engine : Colonia a factorio-like highly optmized.
